# Chromatic

Chromatic is a portfolio and visual engine built around persistent points, surfels, and point-cloud-like rendering.

The product has two layers:

- an experiential layer: train compartment, character, objects, atmosphere
- a functional layer: a readable website carrying the same content

Both should share a point-based visual language on dark grounds with stable motion and restrained glow.

## What This Repo Should Produce First

A working proof of concept:

- one Blender/glTF asset
- one point renderer
- black background
- per-point color
- tunable size and density
- restrained bloom

If that is not compelling in motion, the broader product should wait.

## Product Principles

- The runtime is the platform.
- The site consumes the runtime; it does not define it.
- The train world still matters as product framing.
- Accessibility-critical UI stays DOM-native.
